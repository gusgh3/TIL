# Git command

git 기본 명령어를 정리해봅시다

## init
- 현재 위치에 `.git` 폴더를 생성

```bash
git init
```

## add
- working directory => staging area 옮기는 역할의 명령어

```bash
git add .
```
## status
- 현재 git 상태 확인하는 명령어

```bash
git status
```

## commit
- staging area에 올라간 내용을 스냅샷으로 찍기

```bash
git commit -m "first commit"
```
